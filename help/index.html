Stepwise — URL API

Load routines via URL for AI agents and automation

Usage

Append ?routine=<base64> to the app URL to load a routine directly into the player. The value is a base64-encoded JSON string. The routine is ephemeral — it is not saved and will be gone on the next page load. 

https://<host>/?routine=<base64-encoded-json>

JSON Schema

A routine contains an items array. Each item is either a standalone exercise or a group of exercises that repeats for multiple rounds. 

{
  "name": "string (required, max 50 chars)",
  "items": [
    // Timed exercise:
    {
      "name":     "string (required, max 60 chars)",
      "type":     "time",
      "duration": number (seconds, 5–600)
    },
    // Rep-based exercise:
    {
      "name":     "string (required, max 60 chars)",
      "type":     "reps",
      "reps":     number (1–200)
    },
    // Pause (timed break):
    {
      "name":     "string (required, e.g. 'Pause')",
      "type":     "pause",
      "duration": number (seconds, 5–600)
    },
    // Exercise group (repeats all exercises for N rounds):
    {
      "type":      "group",
      "name":      "string (required, max 50 chars)",
      "rounds":    number (2–20),
      "exercises": [ /* same exercise/pause formats as above */ ]
    }
  ]
}

No id field needed — one is generated automatically. At least one item is required. Three exercise types are supported: time (countdown), reps (manual completion), and pause (rest break with countdown). The legacy "exercises" key (flat array) is still accepted for backwards compatibility.

Example — Simple (no groups)

{
  "name": "Quick Stretch",
  "items": [
    { "name": "Neck Rolls",        "type": "reps", "reps": 10 },
    { "name": "Pause",             "type": "pause", "duration": 5 },
    { "name": "Hamstring Stretch",  "type": "time", "duration": 30 },
    { "name": "Pause",             "type": "pause", "duration": 10 },
    { "name": "Deep Breathing",     "type": "time", "duration": 60 }
  ]
}

Example — With Groups

{
  "name": "Circuit Training",
  "items": [
    { "name": "Warm Up",  "type": "time", "duration": 300 },
    {
      "type": "group",
      "name": "Main Circuit",
      "rounds": 3,
      "exercises": [
        { "name": "Push Ups",  "type": "reps", "reps": 10 },
        { "name": "Pause",     "type": "pause", "duration": 10 },
        { "name": "Pull Ups",  "type": "reps", "reps": 8 },
        { "name": "Pause",     "type": "pause", "duration": 10 },
        { "name": "Abs",       "type": "time", "duration": 30 },
        { "name": "Pause",     "type": "pause", "duration": 60 }
      ]
    },
    { "name": "Cool Down", "type": "time", "duration": 120 }
  ]
}

Base64 encode and append to URL:

const url = baseURL + '?routine=' + btoa(JSON.stringify(routine));

For AI Agents

To create and launch a routine:

  1. Build a routine JSON object matching the schema above
  2. Base64-encode the JSON string
  3. Open <app-url>?routine=<encoded> in the user's browser

The player starts immediately — no user interaction needed beyond the initial page load.
